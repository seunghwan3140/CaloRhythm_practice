Configuration Guide
===================

User Configuration
------------------
Users may configure:

- Daily calorie goal  
- Macronutrient ratio (C/F/P)  
- Custom target nutrients for meals  

Developer Configuration
-----------------------
Developers can modify:

- MySQL database credentials  
- Nutrition dataset paths  
- Default optimization parameters  

Requirements
------------
All dependencies are defined in ``requirements.txt`` :contentReference[oaicite:4]{index=4}.
