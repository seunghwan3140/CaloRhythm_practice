Maintenance and Troubleshooting
===============================

Common Issues
--------------

**Streamlit does not launch**
- Confirm Streamlit is installed  
- Run ``pip install -r requirements.txt``  

**Database connection error**
- Check MySQL service status  
- Validate host/user/password  
- Ensure schema and tables exist  

**Incorrect nutrient results**
- Verify database entries  
- Ensure quantities are numeric  

Recommended Maintenance
------------------------
- Update nutrient database regularly  
- Keep dependencies up to date  
- Follow branch strategy described in Contributing.md :contentReference[oaicite:5]{index=5}
