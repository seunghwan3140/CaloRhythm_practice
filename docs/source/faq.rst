FAQ
===

**Is CaloRhythm free to use?**  
Yes. It is fully open source under Apache 2.0.

**What database is required?**  
MySQL is used for nutrition data storage.

**Can I contribute?**  
Absolutely. Submit Issues or Pull Requests anytime.

**Where can I discuss new ideas?**  
Use the Discussions tab on GitHub.

**Does the project support AI features?**  
Not yet, but future versions will integrate machine learning models.
