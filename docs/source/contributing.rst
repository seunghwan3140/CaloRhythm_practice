Contribution Guidelines
=======================

CaloRhythm welcomes contributions of all types, including bug fixes, feature additions,
and documentation improvements.

Contribution Workflow
---------------------
- Read the Code of Conduct :contentReference[oaicite:6]{index=6}  
- Follow the branch strategy defined in Contributing.md :contentReference[oaicite:7]{index=7}  
- Open Issues for bugs and feature requests  
- Submit Pull Requests with clear commit messages  

Branch Strategy Summary
-----------------------
After each release:

- ``feature/*`` branches are created for new features  
- ``fix/*`` branches handle bug fixes  
- ``docs/*`` branches update documentation  

All branches undergo review before merging.

Pull Request Rules
------------------
- Explain your modifications clearly  
- Reference the Issue when applicable  
- Ensure compatibility with main branch
